# Users CRUD

This app implements a user's CRUD. The backend is emulated with the API with jokes about Chuck Norris

## Content
Covers
```
Basic HTML, CSS & Bootstrap
Webpack
Promises, async/await
Http - fetch - POST/PUT/DELETE
Provider Design Pattern
http Forms
```
